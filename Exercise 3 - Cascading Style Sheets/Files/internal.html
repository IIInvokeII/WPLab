Welcome to Internal/Embedded CSS HTML !

Internal/Embedded CSS is when all styling attributes are declared as selectors within the <style> tag in the <head> section.
The <h1> tag is styled to be blue in colour.
The <em> tag is styled to have a bisque background color.
The <p> tag is styled to have a size of 30px, which is the size you are viewing right now.
